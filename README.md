# 🌿 MWCNU Sewa Gedung

Sistem Informasi Penyewaan Gedung MWCNU (Majelis Wakil Cabang Nahdlatul Ulama)

## 📌 Tentang Proyek

Aplikasi ini adalah sistem manajemen penyewaan gedung berbasis web yang dirancang untuk memudahkan pengelolaan penyewaan gedung di lingkungan MWCNU.

## 🚀 Fitur

- Katalog gedung dengan informasi lengkap
- Booking gedung online
- Upload bukti pembayaran
- Manajemen pemesanan oleh admin
- Dashboard statistik

## 💻 Teknologi

- PHP Native
- MySQL
- HTML5, CSS3, JavaScript
- Font Awesome Icons

## ⚠️ Catatan

**Repository ini adalah versi demo statis untuk GitHub Pages.**

Untuk menjalankan versi lengkap dengan PHP + MySQL:

1. Clone repository ini
2. Jalankan di XAMPP / Laragon
3. Import database `sewa_mwcnu.sql`
4. Akses `http://localhost/penyewaan_gedung/`
